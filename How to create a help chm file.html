How to create a help chm file

  • Microsoft provides a chm create tool called HTML Help Workshop.
  • Download HTML Help Workshop.
  • After HTML Help Workshop installed, there are some tools in the directory.
    • hhw.exe - A GUI for creating chm documents.
    • hhc.exe - A command line program for creating chm documents.

Create chm files in ten minutes

  • Edit filename.hhp 
    [OPTIONS]
Compatibility=1.1 or later
Compiled file=test.chm        #Specify output chm file
Contents file=filename.hhc    #Specify contens file
Default topic=test.html       #Specify the first page
Display compile progress=Yes
Index file=filename.hhk       #Specify index file
Language=0x404 中文 (台灣)


[FILES]


[INFOTYPES]
  • Edit filename.hhc. The program will translate to hhc format file automatically. 
    test|test.html
<TAB>test1|test1.html
<TAB>test2|test2.html
FAQ|FAQ.html
  • Edit filename.hhk. The program will translate to hhk format file automatically. 
    test|test.html
<TAB>test1|test1.html
<TAB>test2|test2.html
hello|hello.html
<TAB>hello1|hello1.html

Why I don't use hhw to create my chm file?

  • Because when I leave a page after I edit it and go to the next page. The program doesn't mention that please save the page. I lost what I edit before. It's not useful.
  • It doesn't support syntax highlighting.
  • When I'd like to replace a string for all files. I have to open files one by one for replacing old string with new string. It's a boring task.
  • If there are some files don't save, the program will pop up many windows to ask you do you want to save the file?.

The components of chm files

  • filename.hhp - HTML help project file
  • filename.hhc - HTML help contents file
  • filename.hhk - HTML help index file
  • filename.html - HTML documents

What a filename.hhp file looks like

[OPTIONS]
Compatibility=1.1 or later
Compiled file=test.chm
Contents file=filename.hhc
Default topic=test.html
Display compile progress=No
Full-text search=Yes
Index file=filename.hhk
Language=0x404 中文 (台灣)


[FILES]


[INFOTYPES]

For the part of language, if the document is Chinese, the value is 0x404 中文 (台灣). If the document is English , the value is 0x409 英文(美國).

Add search page, Full-text search=Yes.
