Vim cheat sheet

Visual mode

Shortcut Description
"*y Copy text to OS clipboard
"*p Paste text from OS clipboard
gj move cursor down (multi lines text)
gt/gT switch tab
p paste to before current line
P paste to after current line
gx open URL under cursor
C-Q Block selection
C-Q then I Insert text before the selection for multiple lines
w Move to the beginning of next word
e Move to the end of next word
[s Move to previous wrong spell
]s Move to next wrong spell
z= Show all spell suggestions. Enter number of suggestions to correct word
gt go to next tab
gT go to previous tab

Split Window

Shortcut Description
C-W P Go to previous window
C-W j/k go to up or down window
C-W h/l go to left or right window
C-W p go to previous window
C-W = set an equal size to the split windows
C-W +/- increase/decrease height
C-W >/< increase/decrease width
C-W _ maximize height
`C-W ` maximize width

Command mode

Shortcut Description
:!start "%" Open file with the default program
:!cmd /k "%" Open file with the default program
:set enc? Show file encoding
:sort Sort lines
:pwd Show current directory
:colorscheme desert Change color scheme
:wa write all open tab files
:set autochdir auto change directory
:!git add .&&git commit Use git commit
:!git push Use git push
:mksession ~/session.vim save session to file session.vim
:source ~/session.vim load session file session.vim
:!del % delete current file on Windows
:!rm % delete current file on Linux
:g/pattern/d delete all lines containing a pattern
:vimgrep /vim/ *.wiki Search word vim in wiki files.
:vimgrep /\<vim\>/ *.wiki Search full word vim in wiki files.
:cfirst Move to the first match.
:clast Move to the last match.
:cprevious or cp Move to the previous match.
:cnext or cn Move to the next match.
:cl List all matches.
:r!dir /w Insert command output to text.
:silent !<command> Prevent "press Enter to continue" message on command prompt
:sp split window
:vsp vertical split window

Search mode

Shortcut Description
/i find any word contains character i
/\<word\> find whole word match word
n move to next match
p move to previous match

Other

Shortcut Description
ctrl-] follow a link in help
