How to install hadoop on Windows 10

  • Download jdk installer from this link.
  • Install jdk-18_windows-x64_bin.msi.
  • Start a new command prompt to test jdk javac -version.
  • Go to hadoop. Click the download button.
  • Download and extract the latest hadoop 3.2.3 binary version.
  • Right click the pc icon in explorer then click properties menu iteom.
  • Type env in the searchbox and hit enter key.
  • Add key HADOOP_HOME and value D:\ZHIHAU\hadoop-3.2.3\bin to user variables.
  • Add key JAVA_HOME and value C:\Program Files\Java\jdk-18.0.1.1\bin to user variables.
  • Add path D:\ZHIHAU\hadoop-3.2.3\bin and C:\Program Files\Java\jdk-18.0.1.1\bin to user PATH variable.
  • Edit hadoop-3.2.3\etc\hadoop\core-site.xml. Paste the following code between configuration block. 
    	 <property>

       <name>fs.defaultFS</name>

       <value>hdfs://localhost:9000</value>

   </property>
  • Edit hadoop-3.2.3\etc\hadoop\mapred-site.xml. Paste the following code between configuration block. 
    <property>

       <name>mapreduce.framework.name</name>

       <value>yarn</value>

   </property>
  • Create folder hadoop-3.2.3\data
  • Create folder hadoop-3.2.3\data\datanode
  • Create folder hadoop-3.2.3\data\namenode
  • Edit hadoop-3.2.3\etc\hadoop\hdfs-site.xml. Paste the following code between configuration block. 
    
<property>

       <name>dfs.replication</name>

       <value>1</value>

   </property>

   <property>

       <name>dfs.namenode.name.dir</name>

       <value>/hadoop-3.2.3/data/namenode</value>

   </property>

   <property>

       <name>dfs.datanode.data.dir</name>

       <value>/hadoop-3.2.3/data/datanode</value>

   </property>
  • Edit hadoop-3.2.3\etc\hadoop\yarn-site.xml. Paste the following code between configuration block. 
    <property>

                <name>yarn.nodemanager.aux-services</name>

                <value>mapreduce_shuffle</value>

   </property>

   <property>

               <name>yarn.nodemanager.auxservices.mapreduce.shuffle.class</name> 

                <value>org.apache.hadoop.mapred.ShuffleHandler</value>

   </property>
  • Edit hadoop-3.2.3\etc\hadoop\hadoop-env.xml. Find JAVA_HOME in this file and modify it. If your java path contains spaces , rename it to short path. Reference 
    set JAVA_HOME="C:\Progra~1\Java\jdk-18.0.1.1"
  • Download and extract https://github.com/s911415/apache-hadoop-3.1.0-winutils.
  • Copy folder bin and replace existed folder bin hadoop-3.2.3\bin.
  • Format namenode via hadoop-3.2.3\bin> hdfs namenode –format
  • Change directory to hadoop-3.2.3\sbin. Execute command start-yarn.cmd. There are many error message.
