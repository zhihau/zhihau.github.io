# git diff
[[git]]

## View changes
| Command                |                                      Description |
|------------------------|-------------------------------------------------:|
| `git diff`             | See all the diff in tracked files but not staged |
| `git diff <file-name>` |                                See a file's diff |
It like `git diff <current-file> <last-version-of-file>`.
