vimwiki Cheat Sheet

Search Shortcut Description
:VimwikiSearch /word/ find any word contains word substring
:VimwikiSearch /\<word\>/ match whole word word
:VimwikiSearchTags vim find vim tag
:lopen Open all match records
:lclose Close all match records window
:lnext Open next match record
:lprevious Open previous match record
Normal Mode Shortcut Description
Shift+Enter Open wiki link in split window
Ctrl+Enter Open wiki link in vertical split window
TAB Go to next link
Shift+TAB Go to previous link
Command Mode Shortcut Description
:h vimwiki-syntax View vimwiki syntax
:VimwikiTOC Generate TOC at first line of current wiki file.
:Vimwiki2HTML Generate html file from current wiki file.
:VimwikiAll2HTML Generate html files from all wiki files.
:h VimwikiSearch vimwiki search help.
Content Description
= Contents = With :VimwikiTOC. It will generate a list of headers under the Contents header.
%nohtml Define don't generate html file in current wiki page.
%%comment If you use nohtml tag, it will generate a broken link on the html page. Convert the broken link to a comment will prevent this problem.
%title my title Define my title for a wiki page.
vimrc command Description
let g:vimwiki_list = [{'path': '~/my_site/', 'auto_toc': 1}] automatically update TOC when the current wiki page is saved.

          \ 'template_default': 'default',
          \ 'template_ext': '.tpl'}]

How to fix error e513 write error_conversion failed?

When I written a file, an error message shows at the bottom of the window. This is a file encoding error. Because vim can written file with UTF-8 file encoding, if this file encodes in UTF-8. file encoding as UTF-8 => Vim sets written file in UTF-8 encode. file encoding as Unknown => Vim sets written file in UTF-8 encode. It will generate below error message. Because mismatch file encoding.

How to fix? Add this line to vimrc file. The line means that open file in UTF-8 encode as default mode. If Vim doesn't know file encode, use one of the fileencodings list to open the file. You can add many file encoding to the fileencodings list. If you'd like to know the file encoding of a file, you can run this command set enc? in Vim command mode. set encoding=utf-8 fileencodings=ucs-bom,utf-8,cp936,gbk,gb18030,gb2312
