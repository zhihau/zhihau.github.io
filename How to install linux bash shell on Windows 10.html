How to install linux bash shell on Windows 10

I want to build vlang code in real bash shell. Not a bash like shell. For instance, MinGW. Because I get some error when I compile vlang code in MinGW.

Steps

  1. Type control panel hit Enter key in the search box.
  2. Click control panel menu item to open it.
  3. Click programs item
  4. Find and click Turn windows features on or off at the right side of the window.
  5. Checked these features Virtual Machine Platform and Windows Subsystem for Linux and click OK button.
  6. Wait a minute.
  7. Reboot your system.
  8. Type microsoft store hit Enter key in the search box. Search Ubuntu in Microsoft store window. Click Get button to download it.
  9. When I download completed, I get a bad user experience. The dark blue button move to a wrong position, it is out of the window.
  10. What's the button? Let's drag the right border of the window to left. We can find the answer. This is a open button.
  11. Click the open button. It will install Ubuntu on your computer.
  12. Afterward you have to set your username and password. Now you have a real bash shell on Windows. Right click on Ubuntu icon in the taskbar. Choose Pin to taskbar to access bash shell quickly.

Ubuntu bash shell environment

  • Where is c and d drive? /mnt/c , /mnt/d
  • How to open file manager on bash shell? Type command explorer.exe ..
  • How to use v command on bash shell? Type command v.exe ..
  • How to run a command in current directory? Type command ./file.exe.
  • If you cannot create a file. When you open a bash, click run as administrator to open it with administrator permission.
  • How to add my bin path to Linux PATH?

Creating a bin folder in the drive d. Copy file.exe to the new directory /mnt/d/bin. Open ~/.bashrc via the command vi ~/.bashrc. Append `export PATH:$PATH:/mnt/d/bin at the end of the file. Then open a new bash shell. Now you can use the file command with file.exe. Remember that the file extension muse have.

How to install an app?

Click start menu. Look for the program then right click. Select uninstall. It doesn't have any uninstalling message to remind you. You need to wait a minute. Then the app icon will disappear. It's not user friendly.

If you opened a Microsoft Store window, a open button shows on the app page. You have to close the Microsoft Store window then open it again. The open button will replace with install button on the app page.It's not user friendly.
